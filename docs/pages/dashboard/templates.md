# Templates — Gaya Intelligence Platform

## Product

**Purpose:** Library of reusable event blueprints that let coaches create new events from proven configurations with a single click, eliminating repetitive setup and surfacing performance benchmarks.

**Target users:** Coaches and operators who run recurring event types and want to standardize their offerings.

**Key features:**
- Template card grid with type filter (All / Camp / Clinic / Private / Tournament) and text search
- Favorite flag (heart icon) and AI-suggested badge (sparkle icon) on cards
- Selected template detail panel shows full configuration: age range, duration, capacity, price, schedule, and tags
- Performance stats panel for templates with prior usage: times used, average fill rate, average revenue
- "Use Template" CTA (gradient button) creates a new event from the selected blueprint
- Edit button for modifying an existing template
- Four summary KPI cards at the top: Total Templates, Times Used, Avg Fill Rate, AI Suggested count
- AI-suggested templates (generated by the platform) are clearly badged and have a distinct border color

**User flows:**
1. Browse the template grid; filter by type or search by name
2. Click a card to load the detail panel on the right
3. Review configuration and historical performance stats
4. Click Use Template to pre-fill a new event creation form
5. Optionally click the edit icon to modify the template before use

## Technical

**Component:** `TemplatesPage` (default export) — `'use client'`

**Imports / dependencies:**
- `useState` from React
- `lucide-react`
- `@/lib/utils` — `cn`

**Key type:**
```ts
interface Template {
  id: string; name: string; description: string;
  type: 'camp' | 'clinic' | 'private' | 'tournament';
  ageRange: string; level: string; duration: string;
  capacity: number; price: number;
  timesUsed: number; avgFillRate: number; avgRevenue: number;
  lastUsed: string; createdBy: string; tags: string[];
  isFavorite: boolean; isAISuggested: boolean;
  schedule: { day: string; time: string }[];
}
```

**Mock data:** `templates` — 6 items; types: camp (2), clinic (2), private (1), tournament (1); one AI-suggested; three favorited.

**State:**
- `search: string` — text filter applied to `t.name`
- `typeFilter: string` — one of `'all' | 'camp' | 'clinic' | 'private' | 'tournament'`
- `selectedTemplate: Template | null` — initialized to `templates[0]`; drives the detail panel

**Derived data (inline):**
- `filtered` — templates matching both `search` and `typeFilter`

**UI patterns:**
- Layout: `grid grid-cols-12`; template grid = `col-span-7`; detail panel = `col-span-5`
- Type filter buttons use `capitalize` on the raw type string
- `typeColors` lookup object maps type to Tailwind badge classes
- Performance stats section only renders when `selectedTemplate.timesUsed > 0`
- "Use Template" button uses a `bg-gradient-to-r from-cyan-500 to-atlas-500` CTA style unique to this page
